## Description
This program is a console-based mini-ERP that allows the management of products, customers, employees, and sales with full CRUD operations, statistics, and JSON-based persistence.

## Features
- Product management (add, edit, delete, search, filters)
- Customer management with purchase history
- Employee management with salary statistics
- Sales system with automatic stock updates
- Reports and analytics
- JSON file persistence

## Data Storage
All data is stored in a single JSON file:
"Administrador de datos.json"

## Structure
- Classes: Product, Client, Employee, Sale, Person
- System controller: Sistema
- Data manager: DataManagerJson

## How to run
Compile using a C++11 compatible compiler with Ejecutar.bat
or otherwise open Parcial de Programacion.exe
